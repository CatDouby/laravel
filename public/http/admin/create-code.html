创建命令 中文名称 参数 操作 描述
make:controller 创建控制器 ->控制器 创建一个新的控制器
make:model 创建模型 ->模型 创建一个新的模型
make:migration 创建迁移文件 ->文件名
->数据表 		创建一个创建迁移文件
make:seeder 创建数据填充文件 ->文件名

make:console 创建artisan命令 ->文件名 创建一个artisan命令
make:auth 创建用户认证 无 创建最基础的用户认证,包括认证相关路由,视图,数据表等

->回滚所有迁移并且再执行一次 		回滚迁移
